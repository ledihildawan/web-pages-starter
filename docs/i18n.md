# i18n System Guide

Runtime internationalization with 42 locales, auto RTL, native digits, and locale-specific formatting.

---

## Quick Reference

### The Golden Rule: `| safe` Filter

```njk
<!-- ✅ HTML output → needs | safe -->
{{ i18n.t('key') | safe }}
{{ i18n.html('key') | safe }}
{{ i18n.formatNumber(val) | safe }}

<!-- ❌ Attributes → NO | safe -->
alt="{{ i18n.text('key') }}"
{{ i18n.attr('key', 'placeholder') }}
```

**Why?** All i18n functions return `<span data-i18n="...">` for runtime translation. `| safe` prevents Nunjucks from escaping the HTML.

## Template API (Complete)

### Translation

| Function | Output | Example |
|----------|--------|---------|
| `i18n.t('key')` | `<span data-i18n="key">` | `{{ i18n.t('page.title') | safe }}` |
| `i18n.t('key', {v})` | `<span data-i18n-vars='{"v":...}'>` | `{{ i18n.t('page.welcome', {name: 'John'}) | safe }}` |
| `i18n.html('key')` | `<span data-i18n="key">` (HTML content) | `{{ i18n.html('page.rich') | safe }}` |
| `i18n.plural('key', n)` | `<span data-i18n-plural="key" data-i18n-count="n">` | `{{ i18n.plural('item', 5) | safe }}` |
| `i18n.text('key')` | Plain string | `alt="{{ i18n.text('page.alt') }}"` |
| `i18n.attr('key', 'attr')` | Attribute string | `{{ i18n.attr('page.ph', 'placeholder') }}` |

### Formatting (All return `<span>`)

```njk
{{ i18n.formatNumber(1234.56) | safe }}              <!-- 1,234.56 -->
{{ i18n.formatCurrency(99.99, 'USD') | safe }}         <!-- $99.99 -->
{{ i18n.formatPercent(0.85) | safe }}                <!-- 85% -->
{{ i18n.formatBytes(1536000) | safe }}               <!-- 1.46 MB -->
{{ i18n.formatDate(new Date()) | safe }}              <!-- 6/2/26 -->
{{ i18n.formatTime(date, 'short') | safe }}           <!-- 1:30 PM -->
{{ i18n.formatDuration(3661) | safe }}               <!-- 1 hour 1 minute -->
{{ i18n.formatOrdinal(1) | safe }}                   <!-- 1st -->
{{ i18n.formatCardinal(1) | safe }}                   <!-- one -->
{{ i18n.formatScientific(1230) | safe }}             <!-- 1.23E+3 -->
{{ i18n.formatAbbreviated(1500000) | safe }}          <!-- 1.5M -->
{{ i18n.formatList(['A', 'B', 'C']) | safe }}         <!-- A, B, and C -->
{{ i18n.formatRelativeTime(-1, {unit: 'day'}) | safe }} <!-- 1 day ago -->
{{ i18n.convertCurrency(100, 'EUR') | safe }}         <!-- €92.50 -->
{{ i18n.formatLocalPrice(plan) | safe }}             <!-- Auto currency by locale -->
{{ i18n.nativeDigits('123', true) | safe }}          <!-- ٠١٢٣ for Arabic -->
```

### Options

```njk
<!-- Raw value (no HTML) -->
<p>{{ i18n.formatNumber(1234, {raw: true}) }}</p>  <!-- Just "1234" -->

<!-- Force digit type -->
{{ i18n.t('page.price', {native: true}) | safe }}     <!-- Native: ٠١٢٣٤ -->
{{ i18n.t('page.price', {universal: true}) | safe }}  <!-- Universal: 0-9 -->

<!-- Custom class -->
{{ i18n.t('page.highlight', {className: 'text-red'}) | safe }}
```

## File Structure

```
src/locales/{locale}/           # 42 locales: id, en-US, ar-SA, ja, zh-CN, etc.
├── common.json5                # Shared: site_name, copyright, etc.
├── components/                 # Component-specific
│   ├── cta.json5              # Used by i18n.t('comp.cta.*')
│   ├── footer.json5
│   └── ...
├── {page}.json5               # Page-specific: hero, pricing, etc.
│   ├── home.json5             # Used by i18n.t('page.*')
│   ├── about.json5
│   └── ...
```

**Key resolution:**
- `page.key` → `{page_id}.json5:key`
- `comp.name.key` → `components/{name}.json5:key`
- `common.key` → `common.json5:key`

## 42 Supported Locales

```
Indonesian:    id, id-ID
English:       en-US, en-GB, en-AU, en-CA, en-IN, en-NZ, en-ZA
German:        de-DE, de-AT, de-CH
French:        fr-FR, fr-BE, fr-CA, fr-CH
Spanish:       es-ES, es-AR, es-CO, es-MX, es-PE, es-PT
Portuguese:    pt-BR, pt-PT, pt-AO, pt-MZ
Arabic:        ar, ar-SA, ar-AE, ar-EG, ar-MA, ar-TN (RTL + Native Digits)
Chinese:       zh-CN, zh-HK, zh-SG, zh-TW
Korean:        ko-KR, ko-KP
Japanese:      ja, ja-JP
Hindi:         hi-IN, hi-NP (Native Digits)
Russian:       ru-RU
```

**Fallback:** `zh-SG → zh-CN → id`

## Common Patterns

### Text with Variables

```njk
<!-- Template -->
<p>{{ i18n.t('page.welcome', {name: userName}) | safe }}</p>

<!-- JSON5 (all locales) -->
{
  "welcome": "Hello, {{name}}!"           // English
  "welcome": "Halo, {{name}}!"            // Indonesian
  "welcome": "مرحبا {{name}}"              // Arabic
}
```

### Pluralization

```njk
<!-- Template -->
<span>{{ i18n.plural('item', count) | safe }}</span>

<!-- JSON5 (English - 2 forms) -->
{
  "item_one": "1 item",
  "item_other": "{{count}} items"
}

<!-- JSON5 (Indonesian - 1 form) -->
{
  "item_other": "{{count}} item"
}

<!-- JSON5 (Arabic - 6 forms) -->
{
  "item_zero": "٠ عنصر",
  "item_one": "عنصر واحد",
  "item_two": "عنصران",
  "item_few": "{{count}} عناصر",
  "item_many": "{{count}} عنصر",
  "item_other": "{{count}} عنصر"
}
```

### Regional Pricing

```njk
<!-- Template -->
<span class="price">{{ i18n.formatLocalPrice(plan) | safe }}</span>

<!-- page.json5 -->
{
  "plans": {
    "basic": {
      "pricing": {
        "base": 29,          // Fallback (USD)
        "id": 500000,       // Indonesia: Rp500.000
        "ja-JP": 4500,      // Japan: ¥4,500
        "ar-SA": 108.75     // Saudi: 108.75 SAR
      }
    }
  }
}
```

### HTML Content

```njk
<!-- Template -->
<div>{{ i18n.html('page.about.html') | safe }}</div>

<!-- JSON5 -->
{
  "html": "Learn about <span class='text-violet-400'>our features</span> and <strong>benefits</strong>."
}
```

## Troubleshooting

| Symptom | Cause | Fix |
|---------|--------|-----|
| `<span data-i18n="...">` visible | Missing `| safe` | Add `| safe` filter |
| `&lt;span&gt;` visible | Used `| safe` on attribute | Remove `| safe` from `i18n.text()` |
| `[page.key]` showing | Key missing from locale | Add key to all 42 locale files |
| Not updating on language switch | Cache or missing key | Check console, hard refresh |
| Wrong currency format | Not using formatter | Use `formatCurrency()` not raw string |
| RTL layout broken | Using physical CSS properties | Use `margin-inline-start` not `margin-left` |

## Best Practices

### ✅ DO

```njk
{{ i18n.t('page.hero.title') | safe }}
alt="{{ i18n.text('page.image_alt') }}"
{{ i18n.formatCurrency(price, 'USD') | safe }}
```

```css
.sidebar { margin-inline-start: 1rem; }  /* RTL-aware */
```

```json5
{
  "copyright": "© 2026"  /* Unicode directly */
}
```

### ❌ DON'T

```njk
{{ i18n.t('page.title') }}                    <!-- Missing | safe -->
alt="{{ i18n.text('alt') | safe }}"          <!-- Wrong: | safe on attr -->
{{ i18n.t('page.price') }}                   <!-- Wrong: use formatter -->
<button>Submit</button>                      <!-- Wrong: hardcoded -->
```

```css
.sidebar { margin-left: 1rem; }  /* Not RTL-aware */
```

```json5
{
  "copyright": "&copy; 2026"  /* Wrong: HTML entity */
}
```

## How It Works

```
┌─────────────────────────────────────────────────────────────┐
│ BUILD (Server)                                                │
├─────────────────────────────────────────────────────────────┤
│ 1. Nunjucks renders template with i18n calls                 │
│ 2. Each call returns <span data-i18n="key">...</span>        │
│ 3. | safe prevents HTML escaping                               │
│ 4. Script injected with window.__I18N_DATA__                 │
├─────────────────────────────────────────────────────────────┤
│ RUNTIME (Client)                                              │
├─────────────────────────────────────────────────────────────┤
│ 1. i18next loads from window.__I18N_DATA__                   │
│ 2. Processes all [data-i18n] elements                        │
│ 3. Updates text content (not HTML structure)                 │
│ 4. Toggles RTL class on <html>                               │
└─────────────────────────────────────────────────────────────┘
```
